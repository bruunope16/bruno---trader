# 🤖 BRUNO TRADER - Deploy no Render

## 📋 PASSO A PASSO COMPLETO

### 1️⃣ CRIAR CONTA NO RENDER (GRÁTIS)

1. Acesse: https://render.com
2. Clique em "Get Started for Free"
3. Cadastre-se com email ou GitHub
4. Confirme seu email

### 2️⃣ FAZER UPLOAD DOS ARQUIVOS

Você tem 2 opções:

#### OPÇÃO A - Via GitHub (Recomendado)

1. Crie uma conta no GitHub (se não tiver): https://github.com
2. Crie um novo repositório:
   - Nome: `bruno-trader`
   - Público ou Privado (tanto faz)
3. Faça upload dos 2 arquivos:
   - `package.json`
   - `server.js`
4. Commit e salve

#### OPÇÃO B - Upload Direto no Render

1. Baixe os 2 arquivos (package.json e server.js)
2. Crie uma pasta no seu PC chamada `bruno-trader`
3. Coloque os 2 arquivos dentro
4. Compacte em .zip

### 3️⃣ CRIAR WEB SERVICE NO RENDER

1. No Render, clique em "New +"
2. Escolha "Web Service"
3. Conecte seu GitHub OU faça upload do .zip
4. Configure:
   - **Name:** `bruno-trader` (ou qualquer nome)
   - **Environment:** `Node`
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Plan:** `Free` ✅

5. Clique em "Create Web Service"

### 4️⃣ AGUARDAR DEPLOY

- O Render vai instalar tudo automaticamente
- Aguarde 2-5 minutos
- Quando aparecer "Live" = Está no ar! ✅

### 5️⃣ ACESSAR SEU ROBÔ

Você vai receber um link tipo:
```
https://bruno-trader-xxxx.onrender.com
```

Abra esse link para ver o status do robô!
